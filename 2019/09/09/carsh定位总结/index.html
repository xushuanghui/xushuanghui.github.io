carsh定位总结

Crash 日志的渠道

  • 苹果收集的 Crash 日志
    • 用户手机上 设置 -> 隐私 -> 分析 里面的，可以连接电脑 Xcode 导出。
    • 在 Xcode -> Window -> Organizer -> Crashes 里面可以查看
  • 自己应用内收集的
    • 接入一些 APM 产品， 如 EMAS、mPaaS、phabricator 等。
    • 接入 PLCrashReporter 、 KSCrash 等 SDK 进行收集，上报到自建平台统计
